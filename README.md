# 🎮 VOID BREAKER

Космический шутер. Уничтожь всех врагов. Выживи.

## Как получить APK

1. Залей этот репозиторий на GitHub
2. GitHub Actions автоматически соберёт APK
3. Зайди во вкладку **Actions** → последний запуск → **Artifacts** → скачай `VoidBreaker-APK`
4. Или зайди в **Releases** — там будет готовый APK

## Установка на телефон

1. Скачай APK
2. Настройки → Безопасность → **Разрешить установку из неизвестных источников**
3. Открой APK → Установить

## Геймплей

- **D-pad** — движение
- **ОГОНЬ** — стрелять вверх
- **БОМБА** — уничтожить всех врагов сразу (редко падает с врагов)
- **🛡️** — щит (1 хит)
- **⚡** — быстрая стрельба
- **💣** — бомба

## Враги

- **👾 Дрон** — быстрый, 1 HP
- **🤖 Танк** — медленный, 3 HP, появляется с уровня 3

## Очки

- Дрон: 100 очков
- Танк: 300 очков
- Комбо множитель за быстрые убийства подряд
